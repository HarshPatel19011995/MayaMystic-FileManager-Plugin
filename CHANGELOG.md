# Changelog

All notable changes to **MayaMystic FileManager** are documented in this file.

This project follows a structured changelog format to track improvements and new features across releases.

---

## [1.0.0] - 2026-05-25

### ✨ Added
- Initial modular FileManager framework architecture
- Centralized `FileManagerService`
- Text file save/load support
- JSON serialization/deserialization support
- Binary file read/write support
- File hashing utilities
  - MD5 hashing
  - SHA256 hashing
- Runtime directory creation utilities
- Configurable file path generation system
- Nested folder path support
- File extension constants system
- ScriptableObject-based configuration system
- `PathUtility`
- `DirectoryUtility`
- `FileReadWriteUtility`
- `FileHashUtility`
- ContextMenu-based runtime testing examples

### 🚀 Improved
- Structured runtime file architecture
- Reusable file path generation workflow
- Cleaner folder organization
- Improved path sanitization
- Better runtime storage scalability

### 🐛 Fixed
- Fixed invalid file name handling
- Fixed unsafe path generation edge cases
- Fixed nested folder path issues
- Fixed inconsistent runtime directory creation handling
